## 📊 About the Project
Sales AI is a full-stack machine learning platform that predicts future sales based on historical data. It supports 7 different business types with industry-specific features.

### 🎯 Key Features
- **7 Business Types**: Retail, E-commerce, Agriculture, Electronics, Food, Manufacturing, Services
- **Multiple Chart Types**: Line, Bar, and Cumulative charts for data visualization
- **AI-Powered Insights**: Business-specific recommendations and action plans
- **Actual vs Predicted**: See how accurate the model is
- **Scenario Planning**: What-if analysis for different business strategies
- **Export Reports**: Download predictions as JSON/CSV
- **Bilingual Support**: Hindi & English interface

### 🛠️ Tech Stack
**Frontend**
- React.js
- Chart.js for visualizations
- CSS3 for styling

**Backend**
- Python Flask
- Scikit-learn (Random Forest, Linear Regression)
- Pandas, NumPy

**Database**
- None (file-based CSV upload)

### 📈 Model Performance
- **Accuracy**: 85-92% depending on data quality
- **Features**: 15+ features including lag variables, rolling averages, seasonal patterns
- **Algorithms**: Random Forest, Linear Regression ensemble
